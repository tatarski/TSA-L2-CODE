Hypertext Markup Language
HTML казва какво е съдържанието на уебсайта
Cascading Style Sheets
CSS показва как уебсайта да изглежда
С JS правим уебсайта интерактивен.


HTML:
- HTML e съставен от тагове
<таг1>АСДФ</таг1>
<таг_без_затварящ>

в html тага е целия HTML документ
в head тага слагаме информация за уебсайта (която няма да се рисува)
в body тага слагаме съдържанието на уебсайта

тагове за head:
<meta charset="UTF-8">
title - заглавие 

тагове за body:
h1,h2,h3.. - заглавие
div - логическо разделение на документа (слага нов ред)

За всеки таг има по един HTML елемент. Повечето HTML елементи се рисуват като правоъгълници.
span - логическо разделение на документа (не слага нов ред)

p - параграф
a - anchor(пренасочване)
img - изображение (може да бъде заредени от интернета/ от компютъра ни)
br - нов ред
hr - хоризонтална линия
button - бутон
b - удебелен текст
i - italics текст
u - underlined текст
canvas - магическо
input - поле за въвеждане
style - таг, в който пишем CSS

<link rel="stylesheet" href="styles.css">  //Зареждаме CSS от файла styles.css

На всеки HTML елемент можем да даваме id. id е уникално име на елемента.

CSS:
В CSS прилагаме правила върху HTML елементи. Тези правила казват как HTML елементите да бъдат нарисувани.

СЕЛЕКТОР {
    ПРАВИЛО1,
    ПРАВИЛО2,
    ПРАВИЛО3,
    ....
}

СЕЛЕКТОРИ:
- име на таг
- много тагове
- id на елемент


JS:
Js се зарежда със script таг-а

let zelkaElement = document.getElementById("zelka") - дава ни елемента, който има id равно на zelka
function onZelkaClick() {
    console.log("kliknah na zelkata")
}
// Когато кликна зелката - искам да се извика функцията onZelkaClick
zelkaElement.onclick = onZelkaClick;

// Променяне на съдържанието на HTML елемент
zelkaElement.innerHTML = "NOVO SUDURJANIE";
